Introdução ao JavaScript

JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web. Cada vez que uma página web faz mais do que simplesmente mostrar informação estática para você, como exibir conteúdo que se atualiza em intervalos de tempo, mapas interativos ou gráficos animados, você pode apostar que o JavaScript provavelmente está envolvido.
